# 七日饮食记录工具

一个纯前端、轻量化的 7 天饮食记录网页，支持手机端使用，可导出结构化 JSON 数据供 AI 进行饮食分析。

## 项目特点
- 纯前端实现，无需后端、无需数据库
- 界面简约干净，移动端友好适配
- 七天饮食横向滑动展示，操作简单直观
- 三餐支持多行文本录入，输入内容自动本地保存
- 个人信息内置写死并隐藏，保护隐私
- 一键导出规范 JSON 文件，可直接被 AI 识别分析
- 导出文件名自动包含姓名与日期，便于归档

## 技术栈
- HTML5（页面结构）
- CSS3（样式布局 + 移动端适配）
- JavaScript（交互逻辑、本地存储、文件导出）


## 使用说明
1. 打开 `index.html` 即可运行
2. 左右滑动切换第 1~7 天饮食记录
3. 在对应餐段输入饮食内容，系统自动保存
4. 记录完成后点击「导出 JSON」，确认后下载文件
5. 将导出的 JSON 文件发送给 AI，即可获得个性化饮食分析

## 功能清单
- ✅ 7 天饮食横向滑动卡片布局
- ✅ 早餐、午餐、晚餐多行文本记录
- ✅ 输入实时自动保存到本地（localStorage）
- ✅ 个人信息内置隐藏，不展示在页面
- ✅ 导出结构化 JSON 文件
- ✅ 导出前确认提示
- ✅ 移动端自适应，体验流畅
